# K-Pop Boy Group Heardle

A clone of [Heardle](https://www.heardle.app/) but for K-pop Boy Group songs. Each song is randomly chosen from a selection of boy group title tracks.

## How to Play

1. Listen to the intro of the song
2. Guess the K-pop boy group song from the list
3. You have 6 attempts to get it right
4. Each wrong guess reveals more of the intro

## Access

- Main site: https://kpop.alw.lol/
- Boy Group version: https://kpop.alw.lol/bg/

## Features

- Daily boy group song challenges
- Popular K-pop boy groups including BTS, EXO, SEVENTEEN, Stray Kids, NCT, GOT7, MONSTA X, and more
- Responsive design for mobile and desktop
- Social sharing features

## Technical Details

- Built with vanilla JavaScript
- Uses SoundCloud for audio playback
- Songs are randomly selected from a curated database
- Local storage for game state persistence
